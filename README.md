# Open Notebook

**English | 한국어**

Privacy-first research platform with AI-powered note-taking, supporting Korean and English languages.

## Features

- **AI Chat Assistant** — Context-aware conversations with privacy controls
- **Smart Notes** — Manual, AI-generated, or chat-saved notes  
- **Content Integration** — PDF, YouTube, web articles, audio/video
- **Transformations** — Custom AI processing patterns
- **Search & Ask** — Full-text + semantic search with cited answers
- **Podcast Generator** — Convert notes to audio in multiple languages
